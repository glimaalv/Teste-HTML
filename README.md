# 🌐 HTML Semântico - Guia de Estudos e Anotações

## 📖 Sobre o Projeto
Este é um projeto desenvolvido exclusivamente em **HTML**, criado com o objetivo de consolidar e reforçar os conhecimentos na linguagem base da web. Ele funciona como um repositório de estudos prático, onde compilei minhas principais anotações, conceitos estruturais e exemplos de uso de tags fundamentais.

O foco principal deste desenvolvimento foi deixar de lado a estilização e a interatividade para focar 100% na marcação pura, na acessibilidade e na correta estruturação do conteúdo.

## ✨ Estruturas e Conceitos Aplicados

O projeto foi construído empregando as seguintes ferramentas da linguagem:

* **HTML Semântico:** Utilização de tags estruturais adequadas (como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>` e `<footer>`) para dar significado ao conteúdo, melhorando tanto a interpretação pelos navegadores quanto a acessibilidade.
* **Formulários (`<form>`):** Implementação de estruturas de coleta de dados, utilizando diversos tipos de inputs, botões, labels e validações nativas do HTML5.
* **Listas (`<ul>`, `<ol>`, `<dl>`):** Organização de anotações e tópicos utilizando listas ordenadas, não ordenadas e de definição.
* **Tabelas (`<table>`):** Estruturação de dados tabulares (com `<thead>`, `<tbody>`, `<tr>`, `<th>` e `<td>`) para catalogar atributos e características das tags estudadas.
