Pontos de Atenção:

  • O atributo action sinaliza o que acontecerá quando o usuário der o submit deste formulário
  • A tag form pode ter os atributos get e post que alteram como os dados são enviados para o Back-End
  • Um button possui 3 atributos: button, reset e submit
  • O atributo for liga o label a um input. Assim se o usuário clicar no texto, o cursor irá automaticamente para o input
  • Os colchetes na tag checkbox indicam ao Back-End que todos os checkbox se tratam de uma lista, se não apenas o último seria lido
  • As tags radio precisam do atributo name para que o site identifique que se tratam de um mesmo grupo e possua a característica de manter apenas um selecionado

Estruturação:

Importante para manter a estrutura do HTML bem sólida quando tratamos de textos, arquivos e semântica.

Essa parte é mais perceptível no código e não no resultado final do site, mas é de extrema importância

Pontos de Atenção:

  • Div pode ser utilizado mas para contêineres genéricos e que não possuem uma tag apropriada, para estruturar o site devemos utilizar: header, main, nav, footer...
  • embed é uma tag caindo em desuso pois não carrega peso semântico, o mais correto é utilizar figure para figuras, iframe para carregar janelas externas...
  • span deve ser utilizado para formatar trechos de código onde o CSS não alcança (apenas um trecho de uma frase ou palavra)

Mídias:

Ao se trabalhar com mídias, temos 4 tags principais além do iframe apresentado acima pela aba do Maps, sendo elas:

  • img
  • audio
  • video
  • track

AUDIO, VIDEO e TRACK

  • As tags audio e video devem ter uma tag source dentro dela para indicar onde o arquivo está contido.
  • Possuem um atributo chamado controls que fornece controles básicos para o controle da mídia.
  • O texto colocado abaixo dessa tag serve para quando não há retrocompatibilidade.
  • o atributo track serve para adicionar legendas a um vídeo através de um arquivo de um formato específico

Semântica:

HTML semântico é o mais importante para otimizar o ARIA (Acessibilidade do site) e o SEO (Search Engine Optimization) para tornar o site mais relevante no Google. Além de manter o código limpo e legível.

Para isso devemos utilizar:

  • Sempre as tags mais específicas o possível. EX:
    • USAR: header, footer, main, aside, article...
    • AO INVÉS DE: div, embed, p...
  • Sempre utilizar alternativas que aumentem a acessibilidade e a retrocompatibilidade do site.
  • O ápice do HTML semântico só pode ser atingido com a prática e a pesquisa contínua.
